## ゲーム概要

経営・経済・社会の仕組みを楽しみながら学べる、親しみやすくも本格的な経営シミュレーションゲームです。
カイロソフト系のフレンドリーなUIと、Capitalism Lab・信長の野望のリアルな経営要素を融合させた設計です。

- スマートフォン（縦画面）最適化
- PWA対応（ホーム画面追加・オフライン動作）
- セーブ/ロード機能（LocalStorage + localforage）
- オートセーブ（月次処理時に自動保存）

## 主要機能

### 人事・組織管理

- **3段階採用フロー** — 求人募集（予算選択）→ 面接 → 契約交渉
- **性格システム（5種類）** — すなお / まじめ / お調子者 / 野心家 / 協調的。それぞれ学習効果・チーム相性が異なる
- **能力パラメータ** — 技術力・営業力・企画力・管理力・創造力の5項目をレーダーチャートで可視化
- **役職システム** — 新人 → 一般 → 主任 → 課長 → 部長 → 取締役
- **モチベーション管理** — 0〜100%でパフォーマンスに連動
- **特別候補者（レア人材）** — 全能力+20ボーナスのエリート人材が確率で登場

### 製品開発

- 部署の実力とチーム相性で品質が決定（最低2名必要）
- 製品ライフサイクル管理（成長期 → 成熟期 → 衰退期）
- 研究開発投資、特許競争

### 財務・経営

- 月次決算（売上・経費・利益の自動計算）
- 融資システム（銀行借入と返済管理）
- マーケティング投資による市場シェア拡大

### イベント・競合AI

- ランダムイベント（大口契約・技術革新・経済不況・業界表彰など）
- 競合AI企業（プレイヤーの成長を検知して戦略を変える）
- 動的ニュースシステム（毎ターン業界ニュース生成）
- 倒産警告・ゲームオーバー判定

### UI/UX

- 5タブナビゲーション（概要 / 人事 / 製品 / 市場 / 財務）
- アニメーション効果（フェード・スライド・パルス）
- タッチ最適化（スマートフォン向け）
- レスポンシブデザイン（最大幅400px）

## セットアップ

### 必要環境

- Node.js 18以上
- npm 9以上

### インストールと起動

```bash
# リポジトリをクローン
git clone https://github.com/TTMK7777/business-simulation-game-v2.git
cd business-simulation-game-v2

# 依存パッケージをインストール
npm install

# 開発サーバーを起動（http://localhost:5173）
npm run dev
```

### スマートフォンでの動作確認（LAN内）

```bash
# LAN内のスマホからアクセスできる形で起動
npm run dev:sp
```

### ビルド・プレビュー

```bash
# 本番ビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## ゲームの遊び方

1. ゲームを起動し、会社名と業種を選択して起業
2. 「人事」タブから従業員を採用し、各部署へ配置
3. 「製品」タブで製品開発を発注（開発部2名以上が必要）
4. 月次ターンを進めて売上・利益を確認
5. 競合AI企業との市場シェア争いに勝利し、会社を成長させる
6. 倒産しないよう資金繰りを管理しながら経営拡大を目指す
