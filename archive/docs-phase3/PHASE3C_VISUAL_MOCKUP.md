#### ❌ 現在
```
  ●      ← 青い丸（頭）
```

**視覚的特徴**:
- カジュアルな青フーディ
- 大きなヘッドホン
- リラックスした前傾姿勢
- メガネ（オプション）

## 🎮 インタラクション体験

### 実装前 ❌
```
ユーザーの操作:
- オフィス画面を開く
- 棒人間が微妙に揺れている
- 「これ何？」と思う
- 職種が分からない
- つまらない
```

### 実装後 ✅
```
ユーザーの操作:
1. オフィス画面を開く
   → 12人のキャラクターが各々動いている

2. 観察する
   → 開発者がタイピング💻
   → 営業が電話中📱
   → 企画がプレゼン準備📊
   → 管理職が書類確認📄

3. ズームイン
   → 表情が見える（笑顔/疲労）
   → 服装のディテールが見える
   → 持ち物が確認できる

4. 時間が経過
   → キャラクターが自動で移動
   → デスク → 会議室 → 休憩エリア
   → 「オフィスが生きている！」

5. ストレスが上がる
   → 😊 → 😐 → 😓 → 💦
   → 表情とエフェクトで状態が一目瞭然

6. キャラクターをクリック
   → 詳細画面でアニメーション再生
   → 愛着が湧く
```
