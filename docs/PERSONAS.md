# ターゲットペルソナ定義

対象: ビジネスエンパイア 2.0（経営シミュレーションゲーム）
策定日: 2026-07-16 ｜ 出自: 外部AIブレスト + 本セッションで正本化
方針: 20〜30代・ビジネス感度高・財務関心層を、購買力・学習動機が異なる3類型に分けて定義する（単一ペルソナでは市場の解像度が落ちるため）。

---

## ペルソナ①：向上心強めの若手ビジネスパーソン「佐藤 隼人（さとう はやと）」

### デモグラフィック
| 項目 | 内容 |
|---|---|
| 年齢 | 26歳 |
| 性別 | 男性 |
| 職業 | 事業会社 経営企画・新規事業企画 |
| 年収 | 480万円 |
| 居住地 | 東京都内（賃貸マンション） |
| 学歴 | 私立大学 経済学部卒 |
| 可処分時間 | 平日夜2時間、休日3〜4時間 |

### バックグラウンド
- 新卒で事業会社に入社し、3年目で経営企画部に異動
- 会議で経営陣の議論についていけず、財務諸表を「読めているつもり」で止まっている
- MBA進学や中小企業診断士取得を検討中だが、時間・費用のハードルで踏み切れていない

### 悩み・ペインポイント
- P/L・B/S・CFの繋がりを体系的に理解しないまま実務をこなしている自覚がある
- 経営判断のシミュレーション経験がなく、「意思決定の勘所」が掴めない
- 座学（本・資格勉強）は退屈で続かない

### ニーズ
- ゲーム感覚で経営数字と意思決定の因果関係を体感したい
- 短時間（隙間時間）でPDCAを回せる設計
- 学んだ内容を実務・転職活動でアピールできる形にしたい

## ペルソナ②：起業志向の20代後半「田中 美咲（たなか みさき）」

### デモグラフィック
| 項目 | 内容 |
|---|---|
| 年齢 | 29歳 |
| 性別 | 女性 |
| 職業 | IT企業 マーケティング職（将来的に独立・起業志向） |
| 年収 | 550万円 |
| 居住地 | 東京都内 |
| 学歴 | 国立大学 商学部卒 |
| 可処分時間 | 平日夜1〜1.5時間、休日は起業準備に充当 |

### バックグラウンド
- マーケティング実務は得意だが、財務・資金繰りなど「経営の裏側」の経験がない
- 副業でスモールビジネスを開始しており、将来的な法人化・起業を視野に入れている
- ビジネス系YouTube・Podcast・書籍を日常的にインプットしている情報感度の高い層

### 悩み・ペインポイント
- 資金調達・キャッシュフロー管理など、起業前に失敗できない領域の実践経験がゼロ
- 経営判断のシミュレーションを「安全に」試せる場がない
- 独学だと知識が断片的で、経営の全体像として繋がらない

### ニーズ
- 資金繰り破綻など、現実ではリスクの高い意思決定を安全に試行錯誤したい
- 起業前提のシナリオ（資金調達、価格戦略、人材採用）を重点的に学びたい
- 学習コミュニティ・SNSでの発信ネタとしても活用したい

## ペルソナ③：管理職手前のプレイヤー「中村 健太（なかむら けんた）」

### デモグラフィック
| 項目 | 内容 |
|---|---|
| 年齢 | 34歳 |
| 性別 | 男性 |
| 職業 | メーカー 営業課長候補（係長） |
| 年収 | 650万円 |
| 居住地 | 首都圏近郊（戸建て・持ち家） |
| 学歴 | 私立大学 理工学部卒 |
| 可処分時間 | 平日夜30分〜1時間、通勤時間中心 |

### バックグラウンド
- 現場のプレイヤーとしては優秀だが、管理職昇進を前に「部門P/L責任」への不安がある
- 会社の管理職研修はあるが座学中心で実践知に乏しいと感じている
- 家庭があり学習時間の確保がシビアなため、効率重視

### 悩み・ペインポイント
- 部門予算・人件費・利益率のトレードオフを扱った経験がない
- 昇進後に「数字が読めない上司」と部下から見られることへの恐怖感
- 忙しく、まとまった学習時間が取れない

### ニーズ
- 通勤時間などスキマ時間で完結する短尺セッション設計
- 自部門クラスの規模感（中小事業部レベル）に近いシナリオ
- 学習成果が社内評価・昇進面談で語れる実績になること

## 3ペルソナ横断の設計インプリケーション

| 論点 | ペルソナ①佐藤 | ペルソナ②田中 | ペルソナ③中村 |
|---|---|---|---|
| 主要モチベーション | 知識体系化・転職市場価値 | リスクフリーな起業練習 | 昇進不安の解消 |
| 求めるシナリオ粒度 | 全社経営 | スタートアップ単体 | 事業部門P/L |
| セッション長の適性 | 中（30分〜1時間） | 長め可（休日集中） | 短（10〜15分） |
| 収益化との相性 | サブスク＋資格連携が刺さりやすい | コミュニティ課金・上位プラン | 法人研修導入（BtoB展開）と相性良 |

**事実と解釈の分離：**
- 事実：3ペルソナとも「座学だけでは経営数字が身につかない」という共通課題を抱える層として設計した
- 解釈（要検証）：ペルソナ③のBtoB研修展開可能性は市場規模が大きい反面、意思決定者が本人ではなく人事部門になるため、CVR設計が別途必要になる可能性が高い
